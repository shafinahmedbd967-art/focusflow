# FocusFlow – Productivity & Focus Web App

FocusFlow is a minimal, distraction-free productivity web application designed to help users stay focused, manage time effectively, and build better work habits.

It combines a Pomodoro focus timer, calming ambient sounds, interactive galaxy background, and productivity analytics to create a relaxing yet powerful focus environment.

---

## ✨ Features

### ⏰ Real-Time Clock

Displays the current time and date synced with the user's local timezone.

### 🧠 Pomodoro Focus Timer

* Start / Stop / Reset timer
* 25 minute focus sessions
* Circular progress indicator
* Session tracking

### 🌌 Galaxy Particle Background

Interactive animated particle galaxy built with HTML Canvas for a modern, immersive UI.

### 🎧 Ambient Sounds

Helps improve concentration with relaxing background audio:

* Rain sound
* Forest sound

### 💬 Inspirational Quotes

Rotating motivational quotes to keep users inspired while working.

### 📊 Focus Analytics

Tracks productivity metrics:

* Total focus sessions
* Focus streak
* Focus activity heatmap

### 🏆 Achievements

Unlock badges based on productivity milestones.

### 🤖 AI Coach

Displays motivational messages after focus sessions to encourage consistent productivity.

### 🎨 Dynamic Themes

Users can switch between different calming gradient themes.

---

## 🧱 Tech Stack

* **HTML5**
* **CSS3**
* **Vanilla JavaScript**
* **Canvas API**
* **Chart.js** (analytics charts)

No frameworks used — built with pure frontend technologies.
